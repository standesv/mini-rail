# Mini Rail Builder V3

Première base jouable inspirée de l'esprit de XTrkCad :
- vue de dessus
- pose libre des rails
- courbes, croisements, aiguillages
- feux rouge / vert
- zones de ralentissement
- départ du train
- sauvegarde locale
- export / import JSON

## Lancer en local

```bash
npm install
npm run dev
```

## Déployer

Le projet peut être poussé tel quel sur GitHub puis déployé sur Vercel ou Netlify.
